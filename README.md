 ## üöÄ About EndlessWiz
 
 **EndlessWiz** is an engaging **English learning platform** designed to make language acquisition enjoyable and effective. Instead of boring drills and grammar rules, users improve their skills by:
 
 - üìñ **Reading interactive stories** with vocabulary highlights and instant translations  
 - üé• **Watching educational videos** with subtitles and comprehension quizzes  
 - üéÆ **Playing language games** that reinforce grammar, vocabulary, and pronunciation

 ## ‚ú® Features
